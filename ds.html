HTML (HyperText Markup Language)

HTML est le langage de balisage standard utilisé pour structurer et organiser le contenu d’une page Web. Il définit la hiérarchie des titres, des paragraphes, des images et des liens, permettant aux navigateurs d’afficher correctement les informations. Facile à apprendre, il constitue la base de toute page Web et s’utilise souvent conjointement avec CSS et JavaScript. Voici un exemple d’un snippet HTML : 

<!DOCTYPE html>
<html lang="fr">
  <head>
    <meta charset="UTF-8">
    <title>Exemple</title>
  </head>
  <body>
    <h1>Bonjour, monde !</h1>
    <p>Ceci est un paragraphe en HTML.</p>
  </body>
</html>

XML (eXtensible Markup Language)

XML est un langage de balisage générique, conçu pour stocker et transporter des données de manière structurée et lisible. Il n’impose pas de balises prédéfinies, ce qui le rend flexible pour créer des formats personnalisés. XML est souvent utilisé pour l’échange d’informations entre applications, la configuration de logiciels ou la description de documents complexes. Exemple de document XML : 

<note>
  <to>Alice</to>
  <from>Bob</from>
  <heading>Salut!</heading>
  <body>Petit message en XML</body>
</note>

JSON (JavaScript Object Notation)

JSON est un format léger d’échange de données, dérivé de la syntaxe JavaScript, mais utilisable par de nombreux langages. Il est particulièrement apprécié pour son aspect lisible et sa simplicité, facilitant la communication entre un serveur et un client (API). JSON est devenu un standard incontournable dans la conception de services web modernes. Exemple de fichier JSON : 

{
  "nom": "Dupont",
  "prenom": "Jean",
  "age": 30,
  "interets": ["Musique", "Sport", "Lecture"]
}

Markdown

Markdown est un langage de balisage léger qui permet de rédiger rapidement du texte mis en forme (titres, listes, liens, images) en utilisant une syntaxe simple. Très populaire sur les plateformes de développement (comme GitHub), il se convertit facilement en HTML. Il permet de se concentrer sur le contenu plutôt que sur la mise en page, tout en restant lisible en brut. Exemple de syntaxe Markdown : 

# Titre de niveau 1

**Texte en gras** et *texte en italique*.

- Liste
- à
- Puces

[Un lien vers un site](https://exemple.com)

LaTeX

LaTeX est un langage de composition de documents reconnu pour sa puissance dans le rendu de formules mathématiques et de textes scientifiques. Il sépare clairement le contenu (texte, formules) de la mise en page et produit des rendus professionnels (articles académiques, présentations). Idéal pour gérer efficacement les bibliographies, index et références croisées, LaTeX est largement adopté dans le monde universitaire et de la recherche. Exemple de document LaTeX : 

\documentclass{article}
\begin{document}

\section{Introduction}
Voici un exemple de texte en \LaTeX.

\begin{equation}
E = mc^2
\end{equation}

\end{document}

CSS (Cascading Style Sheets)

CSS est le langage qui décrit l’apparence visuelle et la mise en page d’un document HTML. Il permet de définir les couleurs, la taille et la police des textes, l’agencement des éléments ou encore les effets visuels et les animations. Grâce à CSS, il est possible de séparer clairement la structure du contenu (HTML) de sa présentation, rendant la maintenance et l’évolution d’un site plus simples. Exemple de code CSS : 

body {
  margin: 0;
  font-family: Arial, sans-serif;
  background-color: #f7f7f7;
}

h1 {
  color: #333;
}

Cette page présente plusieurs langages de structuration (HTML, XML, JSON, Markdown, LaTeX et CSS).
Ces langages sont tous très utiles pour créer et mettre en forme des contenus.

En savoir plus :

HTML

Principales utilisations : Création de la structure de pages web, intégration d’images, de liens, de formulaires et d’éléments interactifs.
Cas d’usage : Sites web vitrine, blogs, email marketing (modèles HTML), etc.

XML

Principales utilisations : Stockage et transfert de données, configuration d’applications, définition de langages spécialisés (par ex. RSS, SVG).
Cas d’usage : Fichiers de configuration (Android, logiciels bureautiques), flux RSS, interopérabilité entre systèmes.

JSON

Principales utilisations : Transfert de données léger entre serveur et client, souvent dans les APIs REST.
Cas d’usage : Réponses d’API (ex. JavaScript/Node.js, Python Flask/Django), stockage de paramètres d’application, fichiers de configuration.

Markdown

Principales utilisations : Rédaction simple et rapide de documents ou de README, blogs statiques, documentation de projets open-source.
Cas d’usage : README GitHub, wiki, publications rapides sur des plateformes de blogging.

LaTeX

Principales utilisations : Création de documents scientifiques, articles de recherche, rapports, présentations (via Beamer).
Cas d’usage : Thèses universitaires, articles de revues, documents nécessitant des formules mathématiques complexes.

CSS

Principales utilisations : Mise en forme et design des pages web : gestion des couleurs, typographies, espacements, animations.
Cas d’usage : Intégration web, thèmes responsives, frameworks CSS (Bootstrap, Tailwind), animations et transitions.
